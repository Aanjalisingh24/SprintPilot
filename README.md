SprintPilot – AI-Powered Jira Management & QA Copilot

SprintPilot is a full-stack MERN application that combines Jira ticket visualization with an AI-powered QA Copilot to help developers, testers, and project teams quickly understand project information and technical concepts. The application provides secure user authentication, displays Jira ticket details through an intuitive dashboard, and features an intelligent AI assistant capable of answering both project-specific and general technical questions.

The AI Copilot uses Groq Llama 3.3 to analyze live Jira ticket data, explain technical terminology, summarize ticket information, and provide real-time streaming responses for a smooth conversational experience similar to ChatGPT.

# Features

* Secure JWT Authentication
* Protected Routes
* View Jira ticket and project information
* AI Copilot powered by Groq (Llama 3.3)
* ChatGPT-like real-time streaming responses
* Context-aware answers based on Jira ticket data
* Answers general programming and software engineering questions
* Explains technical terms, concepts, and Jira ticket descriptions
* Markdown-formatted AI responses
* Responsive UI built with React & Tailwind CSS
* AI Copilot Capabilities

# The AI Copilot can assist users by:

* Answering questions about Jira tickets and project details
* Explaining technical ticket descriptions in simple language
* Identifying ticket priorities, statuses, and assignees
* Summarizing project information
* Explaining programming concepts and technical terminology
* Answering general software engineering questions
* Helping testers understand feature requirements and technical details
* Providing intelligent responses using both Jira ticket context and the model's general knowledge
* Streaming responses in real time for a conversational experience
